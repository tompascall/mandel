Mandelbrot set

This is a tiny application to study the Mandelbrot set. The app gives a graphical representation of the set. You have a lot of color schemes to try. It is possible to enlarge an area of the set by marking out the area with your mouse. You can set the size of the canvas and the value of the maximum iteration. n my experience it is the best choice to run the app on Firefox. Sorry, but your browser does not support the HTML5 canvas tag. Please try the app on an other browser.
